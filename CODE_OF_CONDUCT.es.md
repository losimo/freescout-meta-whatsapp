# Código de conducta

[Català](CODE_OF_CONDUCT.ca.md) · [English](CODE_OF_CONDUCT.md) · [Castellano](CODE_OF_CONDUCT.es.md)

Este proyecto quiere una comunidad abierta y exigente a la vez. La regla general cabe en una frase: trata a las personas con respeto y a las ideas con rigor.

## Se espera

- Buena fe por defecto: asume que la otra persona quiere mejorar el proyecto.
- Crítica técnica directa y argumentada, siempre sobre el código o la propuesta, nunca sobre la persona.
- Aceptar el desacuerdo: las decisiones de mantenimiento se toman con criterio explicado, y un "no" argumentado cierra el tema si no hay argumentos nuevos.
- Respetar el tiempo de los demás: contexto claro, lectura previa y mensajes concretos.

## No se acepta

- Ataques personales, descalificaciones, acoso o discriminación de ningún tipo.
- Insistencia agresiva después de una decisión ya explicada.
- Publicar información privada de otras personas.
- Usar el proyecto como campo de batalla ajeno al mantenimiento del código.

## Ámbito y aplicación

Este código se aplica a issues, PRs, discussions y cualquier otra comunicación vinculada al proyecto.

Los incumplimientos se pueden reportar de manera privada al mantenedor: losimo@gmail.com.

## Medidas

Las medidas serán proporcionales y progresivas: aviso, bloqueo temporal o expulsión permanente. Los casos graves pueden recibir una respuesta inmediata.

## Final

Si tienes un desacuerdo técnico, explícalo con claridad y con datos. Si tienes una queja sobre conducta, comunícala por un canal privado y con la mínima difusión posible.
